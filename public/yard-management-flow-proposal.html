Gate & Yard Management Flow

Proposed Changes & Updated System Flow
Prepared: March 12, 2026

Summary of Changes

Removed

  • Camera capturing truck plate number at gate

Added

  • Driver barcode scanning for check-in
  • System auto-assigns slot to driver
  • Driver self check-in and self check-out
  • Storekeeper role and workflow
  • Storekeeper requests truck for processing
  • Storekeeper marks processing as complete
  • Overstay detection and notifications

Modified

  • Check-in is now driver-initiated (was gate staff)
  • Slot assignment is now automated (was manual)
  • Check-out is now driver-initiated (was gate staff)
  • Vehicle identification via barcode (was camera + license)

⚠ Overstay Detection (Runs Continuously)

If a vehicle exceeds its expected stay duration, the system automatically: (1) Flags the visit as overstay on the dashboard  →  (2) Sends alert notification to the driver  →  (3) Sends alert notification to the yard manager  →  (4) Highlights the slot with a pulsing overstay indicator

Role Responsibilities

Driver

  • Scan barcode at gate to check in
  • Receive auto-assigned slot information
  • Navigate to slot independently
  • View active visit status and timer
  • Self check-out when cleared
  • View scheduled appointments
  • Receive overstay notifications

Storekeeper New Role

  • View all checked-in trucks in yard
  • Filter trucks by purpose (delivery, pickup, etc.)
  • Request a truck for processing
  • Oversee loading/unloading operations
  • Mark processing as complete
  • Notify driver when cleared to leave

Admin / Yard Manager

  • Configure yards, zones, and slots
  • Design yard layout visually
  • Monitor live dashboard in real-time
  • Manage appointments and confirmations
  • Override slot assignments if needed
  • Receive overstay alerts
  • View utilization reports and analytics
